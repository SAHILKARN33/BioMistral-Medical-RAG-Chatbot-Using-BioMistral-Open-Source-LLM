# BioMistral Medical RAG Chatbot Using BioMistral Open Source LLM

This project is a Medical Chatbot built using the **BioMistral Open-Source LLM**, combined with a **Retrieval-Augmented Generation (RAG)** pipeline.  
It allows users to ask medical-related queries and get responses grounded in uploaded medical documents.
